## 🐺 How to run locally

From your command line:

```bash
# Clone this repository
$ git clone https://github.com/feliperodriguess/holograph-tw-clone.git

# Go into the repository
$ cd holograph-tw-clone

# Install the dependencies
$ npm i

# Run locally
$ npm run dev
```
